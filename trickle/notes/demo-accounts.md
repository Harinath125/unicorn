## Usage Instructions

1. Visit the main landing page
2. Click on any demo email in the authentication form to auto-fill credentials
3. Select the appropriate role from the dropdown (auto-selected when clicking demo accounts)
4. Click "Sign In" to access the role-specific dashboard
5. Alternatively, use any custom email and password combination for testing

## System Behavior

- All demo accounts provide immediate access without validation
- User sessions are stored locally for the demo experience
- Role-based redirects ensure users land on the appropriate dashboard
- Cross-role access is restricted based on permissions matrix
